Enlaces dentro de HTML

Los enlaces en HTML se crean utilizando la etiqueta <a> (ancla) junto con el atributo href, que define el destino del hipervínculo.
Estos permiten conectar páginas web, archivos o secciones específicas dentro de un mismo documento. Su sintaxis básica es: <a href="url">Texto del enlace</a>.
